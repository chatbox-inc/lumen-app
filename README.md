# メッセージ管理

## support 

フロントからのメッセージ投げ込みのみをサポート

メッセージの投げ込みにフック(return mail 等で活用)

from to body からなるメッセージ構成(required)

from to をユーザIDにしてSMS活用

to をbox_idにしてBOX MESSAGE活用など

## 利用方法

MessageServiceInterface を注入する
